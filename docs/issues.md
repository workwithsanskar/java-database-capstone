# Smart Clinic User Stories

## Doctor User Stories

- As a doctor, I want to log in securely so that I can access only my portal data.
- As a doctor, I want to view my appointments by date so that I can prepare for consultations.
- As a doctor, I want to create prescriptions after each consultation so that patients receive accurate medication instructions.
- As a doctor, I want to manage my available time slots so that patients can book valid appointments.

## Patient User Stories

- As a patient, I want to register and log in so that I can book and manage appointments.
- As a patient, I want to search doctors by name or speciality so that I can find the right doctor quickly.
- As a patient, I want to book appointments for an available slot so that I can receive care without conflicts.
- As a patient, I want to view my appointment history so that I can track past and upcoming visits.

## Admin User Stories

- As an admin, I want to log in to a secure admin portal so that I can manage clinic operations.
- As an admin, I want to add doctors to the system so that their schedules are available for patient bookings.
- As an admin, I want to manage doctor and patient records so that data remains accurate and complete.
- As an admin, I want to review clinic reports so that I can monitor appointment trends and doctor utilization.
