# Smart Clinic Management System

Smart Clinic Management System is a sample full-stack capstone project that demonstrates a clinic workflow across Admin, Doctor, and Patient portals.

## Project Structure

- `backend/`: Spring Boot REST API
- `frontend/`: HTML, CSS, and JavaScript portal screens
- `docs/`: architecture notes, user stories, schema design, and submission helpers
- `sql/`: schema, seed data, stored procedures, and sample query outputs
- `screenshots/`: submission-ready portal screenshots
- `.github/workflows/`: CI workflow

## Main Features

- Doctor, patient, and admin user stories
- MySQL schema with appointments, prescriptions, doctors, and patients
- JWT token generation and validation
- Appointment booking and doctor availability lookup
- Prescription creation API
- Docker multi-stage build
- GitHub Actions Maven compilation workflow
