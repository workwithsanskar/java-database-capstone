# User Stories

## Doctor

### 1. View Patient Records
- As a doctor, I want to view patient records so that I can understand patient history.
- Acceptance Criteria: Doctor can view patient details.
- Priority: High
- Story Points: 3

### 2. Update Prescriptions
- As a doctor, I want to update prescriptions so that patients receive correct treatment.
- Acceptance Criteria: Doctor can add/update prescriptions.
- Priority: High
- Story Points: 3

---

## Patient

### 3. Book Appointment
- As a patient, I want to book appointments so that I can consult a doctor.
- Acceptance Criteria: Patient can select doctor and time slot.
- Priority: High
- Story Points: 5

### 4. View Medical History
- As a patient, I want to view my medical history so that I can track my health.
- Acceptance Criteria: Patient can see past records.
- Priority: Medium
- Story Points: 3

---

## Admin

### 5. Manage Doctors
- As an admin, I want to add or remove doctors so that the system stays updated.
- Acceptance Criteria: Admin can add/delete doctor records.
- Priority: High
- Story Points: 5

### 6. Manage Appointments
- As an admin, I want to manage appointments so that scheduling runs smoothly.
- Acceptance Criteria: Admin can view and update appointments.
- Priority: Medium
- Story Points: 4
